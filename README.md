# Sprint_7

# Тестирование API для сервиса Самокат

В этом проекте реализованы автоматизированные тесты для ключевых функций API сервиса Яндекс Самокат  

## Проверенные функции

1. **Создание курьера** – проверка возможности регистрации нового курьера.
2. **Логин курьера** – проверка авторизации курьера по логину и паролю, включая негативные сценарии.  
3. **Удаление курьера** – тестирование удаления курьера по ID.  
4. **Создание заказа** – проверка возможности создания нового заказа на самокат с различными параметрами (адрес, цвет, дата доставки и др.).  
5. **Принятие заказа** – проверка возможности курьера принять заказ.  
6. **Получение заказа по номеру (track)** – тестирование запроса заказа по уникальному номеру.  
7. **Получение списка заказов** – проверка получения всех заказов в системе.  


1. Установите зависимости
pip install -r requirements.txt

2. Запуск всех тестов
pytest --alluredir=allure_results

3. Просмотр Allure отчёта
allure serve allure_results
